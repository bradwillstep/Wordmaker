Wordmaker is a **free**, **mobile-friendly**, Word-like editor that runs fully in the browser and can be installed as an app (PWA).

## Features
- Rich text editor (Quill)
- Character skins (bunny/shark/tiger/zebra/lion/cat/dog)
- Calculator dropdown + AI Solve (open-source, in-browser)
- Open & edit: **DOCX, PDF (text+OCR), ODT, PAGES (best-effort), TXT, HTML, RTF**
- Export: DOCX, PDF, Print
- Grammar check (LanguageTool)
- Optional AI writing tools (BYO API key)
- Installable PWA + offline caching
